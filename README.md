# Nord Light

Nord Light:

![](https://github.com/gaojr/vscode-nord-light/raw/master/screenshot.png)

If you want to add bold style for keywords, use this:

```json
{
    "scope": [
        "keyword",
        "storage",
        "constant.language",
        "support.class.builtin",
        "entity.name.tag.html",
    ],
    "settings": {
        "fontStyle": "bold"
    }
}
```
